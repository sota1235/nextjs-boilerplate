====

Boilerplate for using Next.js including base setup.

### Description

Scaffolding for using Next.js with...

- Redux
- ESLint
- prettier
- Jest

### Requirement

- Node.js v12>=

### Usage

- test
  - `npm run test`
- lint
  - `npm run lint`

### Install

```
npm i
npm run dev
```
