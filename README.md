====

Boilerplate for using Next.js including base setup.

### Description

Scaffolding for using Next.js with...

- Redux
- ESLint
- prettier
- Jest
- styled-components
- stylelint

### Requirement

- Node.js
  - See [.node-version](.node-version) to confirm supported version.

### Usage

See result of `npm run`.

### Install

```
npm i
npm run dev
```
